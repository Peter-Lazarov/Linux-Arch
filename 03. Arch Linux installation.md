# 2. Дискове
## Разделяне на дяловете
Нека да видим какво имаме като дискове и дялове с
```bash
lsblk
```

Двам пример с лаптоп Thinkpad T14s Gen1 с ssd 256 GB.  
Инсталирани са вече Windows 11 и Endeavour OS.  
Целта е да се изтрие Endeavour OS и да се инсталира Arch Linux на неговото място.

```bash
# Windows 11 - дялове които остават
    /dev/nvme0n1p1 – EFI (Windows boot)
    /dev/nvme0n1p2 – MSR
    /dev/nvme0n1p3 – BitLocker Windows C:
    /dev/nvme0n1p4 – Windows recovery
# EndeavourOS - дялове които се трият
    /dev/nvme0n1p5 – /efi (на EndeavourOS)
    /dev/nvme0n1p6 – неизвестен (също EndeavourOS)
    /dev/nvme0n1p7 – swap
    /dev/nvme0n1p8 – /
    /dev/nvme0n1p9 – /home
# Arch Linux - дялове които се създават
    / - 60 GB
    /home - 60-70 GB
    swap - 8GB
    EFI - 256-512 MB only if this in the one operating system for the device
```

## Delete partition
### from old Endeavour OS
```bash
sudo fdisk /dev/nvme0n1
p - показва дяловете

d - трие и пита кой номер
5

w - записва промените
```

## Create partitions
```bash
# Root
sudo fdisk /dev/nvme0n1
n - създава дял
5
<default first sector>
+60G

    # Partition 5 contains a vfat signature 
    # remove signature yes or no
    y
w - записва промените    
```
```bash
    Какво е „подпис“ (signature)
        Това е малък маркер в началото на дяла, 
        който казва каква файлова система е била там:
            vfat
            ext4
            ntfs
            Swap
```
```bash	
# Swap
sudo fdisk /dev/nvme0n1
n - създава дял
7
<default first sector>
+8G - за Swap
w - записва промените
```

```bash
# Home
sudo fdisk /dev/nvme0n1
n - създава дял
6
<default first sector>
<default size> - да вземе останалото място
w - записва промените
```
## 2.1 Format the partitions
```bash
/21_Formatting.sh
```
```bash
#!/bin/bash

# / (root)
mkfs.ext4 /dev/nvme0n1p5

# /home
mkfs.ext4 /dev/nvme0n1p6

# Swap
mkswap /dev/nvme0n1p7

# EFI
mkfs.fat -F32 /dev/nvme0n1p1
```

## 2.2 Mounting
```bash
/22_Mounting.sh
```
```bash
#!/bin/bash

# / (root)
mount /dev/nvme0n1p5 /mnt

# /home
mkdir /mnt/home
mount /dev/nvme0n1p6 /mnt/home

# Swap
swapon /dev/nvme0n1p7

# EFI
mkdir -p /mnt/boot
mount /dev/nvme0n1p1 /mnt/boot

# за проверка
lsblk
```

# 3. Инсталиране на Arch Linux
## Много е важно да са mount-нати всичките 4 точки от раздела 2.2 Mounting
Пуска се от инсталатора archiso - root@archiso ~ #
```bash
/31_Installing.sh
```

```bash
#!/bin/bash

# Инсталиране на Arch Linux
pacstrap /mnt \
    base \
    linux \
    linux-headers \
    linux-firmware

# Неща за мрежа, аудио, EC, ACPI
pacstrap /mnt \
    networkmanager \
    iwd \
    dhcpcd \
    sof-firmware

# Неща за звук, bluetooth, енергоспестяване
pacstrap /mnt \
    pipewire \
    pipewire-alsa \
    pipewire-pulse \
    wireplumber \
    bluez \
    bluez-utils \
    tlp \
    tlp-rdw

# Автоматично монтиране на дисковете след инсталацията
# Чете какво си mount-нал в /mnt
# Взима UUID-ите на дяловете
# Пише ги в /mnt/etc/fstab
genfstab -U /mnt >> /mnt/etc/fstab
    # за проверка
    cat /mnt/etc/fstab
```

```bash
# Влизаш в новоинсталираната система върху диска.
arch-chroot /mnt  
    # Arch ISO остава само "инсталатор", а chroot те прехвърля вътре в истинския Arch.
    # Оттук нататък всички команди се изпълняват все едно системата вече е стартирана от диска.
```

# 4. Grub boot loader
## Инсталираме Grub boot loader
```bash
# Влизаме в системата chroot - [root@archiso ~#]
arch-chroot /mnt
```
Пуска се от системата chroot - [root@archiso ~#]
```bash
pacman -Sy grub efibootmgr
grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg

efibootmgr

exit
reboot
```
# 5. Настройки
## 5.1 Регионални настройки
Пуска се от системата chroot - [root@archiso ~#] с
```bash
# Скриптове се пускат с /name.sh
/51_Regional.sh
```

```bash
#!/bin/bash

ln -sf /usr/share/zoneinfo/Europe/Sofia /etc/localtime
hwclock --systohc

sed -i 's/#en_US.UTF-8 UTF-8/en_US.UTF-8 UTF-8/' /etc/locale.gen
sed -i 's/#bg_BG.UTF-8 UTF-8/bg_BG.UTF-8 UTF-8/' /etc/locale.gen
locale-gen

echo "LANG=en_US.UTF-8" > /etc/locale.conf
echo "arch" > /etc/hostname

cat <<EOF > /etc/hosts
127.0.0.1   localhost
::1         localhost
127.0.1.1   arch.localdomain arch
EOF

# Font
pacman -S --noconfirm terminus-font
setfont ter-228n
echo "FONT=ter-228n" > /etc/vconsole.conf

# NetworkManager
systemctl enable NetworkManager

```

## 5.3 Създаване на потребител
Пуска се от системата chroot - [root@arch ~#] с
```bash
/53_User.sh
```
```bash
#!/bin/bash

# Sudo
pacman -S --noconfirm sudo

# User
useradd -m -G wheel peter

# sudo for wheel group
sed -i 's/# %wheel ALL=(ALL:ALL) ALL/%wheel ALL=(ALL:ALL) ALL/' /etc/sudoers

```
Ако точка 4) не проработи, се добавя потребителя ръчно
```bash
usermod -aG wheel peter
EDITOR=nano visudo
# Търси се ред %wheel ALL=(ALL:ALL) ALL
# и се разкоментирва
```
