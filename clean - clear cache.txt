1. Clear the Package Cache - var/cache
sudo pacman -Sc

2. Clean the /tmp Directory
sudo rm -rf /tmp/*

3. Clear User Cache
rm -rf ~/.cache/*

4. Remove Unneeded Packages
sudo pacman -Rns $(pacman -Qtdq)

5. Clean Up AUR Package Cache
yay -Sc

6. Additional cleaning - logs, temporary files
sudo journalctl --vacuum-size=200M
sudo rm -rf /var/tmp/*
sudo rm -rf /tmp/*

7. (dangerous cleaning) Removes ALL cached packages, including the latest ones, cannot downgrade, without downloading it again
sudo pacman -Scc
